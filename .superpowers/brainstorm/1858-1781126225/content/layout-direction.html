Какой главный лайаут подходит продажнику?

Сейчас: таблица → клик → chat-view на весь экран. Три альтернативы:

Кофейня Ромашка — VK
Да, расскажите подробнее
Есть 5 минут созвон?

A — Split Panel (два столбца)

Список лидов слева, активный диалог справа. Никакого переключения экранов. Как Superhuman / Linear.

B — Канбан-доска

Колонки по статусам, карточки лидов. Воронка продаж сразу видна. Как Trello / Pipedrive.

Поиск / команда...
Открыть диалог → Ромашка
Закрыть сделку → Ромашка
Отправить ледокол → Barbershop

C — Command Palette + текущий лайаут

Оставить таблицу+chat, добавить Cmd+K (быстрый поиск и действие без мыши). Минимум переработки, максимум скорости.

Кликни на карточку или напиши в терминале.
