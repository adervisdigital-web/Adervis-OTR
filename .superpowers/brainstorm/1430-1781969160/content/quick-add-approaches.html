A — Quick Bar в сайдбаре

Поле ввода прямо над списком лидов. Вставил ссылку → Enter → лид появился в списке.

Плюсы

  • Нет кликов — ноль переходов
  • Виден всегда

Минусы

  • Занимает место в сайдбаре
  • Нет места для доп. полей

B — Модалка URL-first ⭐ Рекомендую

Клик на + открывает модалку. Вставляешь ссылку — имя и платформа заполняются автоматически. Одно подтверждение.

Плюсы

  • Работает с мобильным FAB
  • Есть место для сегмента
  • Мало кликов (2)

Минусы

  • Один лишний клик vs A

C — Улучшить текущую форму

Автозаполнение имени при вставке ссылки в существующей строке таблицы. Без новых компонентов.

Плюсы

  • Минимум изменений

Минусы

  • Форма только в table-view
  • FAB на мобильном по-прежнему сломан
