ADERVIS OTR — Роадмап

Два этапа: сначала работаем сами, потом продаём

ЭТАП 1
Для вас двоих ~3–5 дней
🔐 Auth — Supabase Login
Экран входа email+пароль. Вы и сотрудник входите через браузер. Сессия сохраняется.
☁️ Облачная база лидов
localStorage → Supabase PostgreSQL. Оба видят одни и те же лиды в реальном времени.
⚡ Real-time sync
Добавил лид — у второго обновляется таблица без перезагрузки. Supabase subscriptions.
👤 Кто работает с лидом
Колонка «Менеджер» в таблице. Фильтр «Мои лиды / Все». Кто создал — тот владелец.
📱 PWA — установка как приложение
manifest.json + service worker. Открывается как мобильное приложение. (Единственный TODO из плана)
ЭТАП 2
SaaS-продукт +1–2 недели
🌐 Лендинг + домен
Сайт «OTR — CRM для холодных продаж». Netlify бесплатный хостинг. Регистрация → создаёт workspace.
💳 Оплата — Stripe / ЮKassa
Подписка ~990–1490 руб/мес за команду до 5 чел. 14 дней бесплатно. Supabase Edge Functions для webhook.
🏢 Мультиworkspace
Каждый клиент = свой изолированный workspace. RLS (Row-Level Security) в Supabase.
📧 Приглашение команды
Владелец вводит email → приходит письмо с ссылкой → сотрудник регистрируется и сразу в workspace.
🎯 Целевая аудитория
Видеостудии, SMM-агентства, фрилансеры-аутричеры. ВКонтакте/Telegram-сообщества продажников.
🔧 Улучшения логики + дизайна (параллельно с Этапом 1)
