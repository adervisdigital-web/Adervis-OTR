Нажми на вариант чтобы выбрать. Можно выбрать несколько.

A
Иконки + дропдаун «Данные»
Компактно, 4 элемента

📁 Данные ▾
⚙️
ℹ️

ℹ️ = Справка

B
Два дропдауна + CTA
Явная группировка по назначению
ADERVIS | OTR

«Отчёты» = Экспорт CSV + Бэкап + Импорт
«Настройки» = Шаблоны + Gemini API + Справка
Итого: 3 элемента, группировка по смыслу
