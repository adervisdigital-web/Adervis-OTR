A — Модалка (улучшенная)

Клик по лиду — большая модалка с полноценным чатом и скриптами снизу. Занимает 70–80% экрана.

Плюсы

  • Минимальные изменения
  • Фокус на одном лиде

Минусы

  • Нет контекста таблицы рядом
  • Медленное переключение

B — Drawer справа (split-view)

Клик по лиду — таблица сужается слева, справа выезжает панель с чатом. Оба видны одновременно.

Плюсы

  • Контекст таблицы виден
  • Быстрое переключение между лидами

Минусы

  • Сложнее в реализации
  • Меньше места для чата

C — Страница чата (как Telegram)

Клик — переход на отдельный экран. Слева список лидов, справа полноценный чат. Максимально похоже на TG.

Плюсы

  • Знакомый UX (TG/WhatsApp)
  • Много места под диалог и AI

Минусы

  • Нужна навигация между экранами
  • Самая большая переработка
