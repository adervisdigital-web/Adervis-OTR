## Что такое ADERVIS | OTR

**OTR = Outreach** — персональная CRM-воронка холодных продаж для компании **ADERVIS** (видеопродакшн).

**Задача менеджера:** находить заведения (кафе, рестораны, барбершопы, салоны и т.д.) в ВКонтакте / Instagram / Telegram и предлагать им съёмку коротких видео (VK Клипы, Reels, Shorts) для привлечения новых гостей.

**Воронка продаж:**
```
Новый лид  ->  Ледокол отправлен  ->  В диалоге  ->  УСПЕХ (сделка)
                                                   ->  Отказ / Игнор
```

**Что помогает закрывать сделки:**
- Готовые скрипты под каждый этап и каждую платформу (VK/Inst/TG)
- Варианты ответов на возражения: «нет бюджета», «есть SMM», «пришлите примеры»
- CTA-плейсхолдеры {call_link}, {brief_link} — подставляются автоматически
- Таймер просроченных (2+ дня без ответа) — напоминание сделать follow-up

## Технический стек

- Тип: Single-file HTML app (Adervis LidGen.html)
- Путь: C:\work\lidgen\Adervis LidGen.html
- JS: Vanilla JS (без фреймворков)
- CSS: CSS Custom Properties, Linear.app design tokens
- Хранилище: localStorage
- Ключи БД: adervis_cold_db_v3 (лиды), adervis_custom_scripts_v4 (скрипты), adervis_cta_v1 (CTA)

Структура лида:
```
{ id, name, link, contact, bizType, status (0-4), updatedAt, notes, clientReply }
```

## Правила работы

1. После каждого изменения обновлять ACTION_MAP.md (статус -> DONE + дата)
2. После каждого шага выдавать промпт для следующего шага
3. Предлагать /clear для экономии токенов после выполнения задачи
4. Не трогать JS-логику если задача только CSS/UI
5. Не добавлять фичи не из плана без явного запроса
